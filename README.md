# hackwoman-back

back-end for the project developed during the provi hack para todos hackathon

## technologies

- nodejs
- mongodb
- express

## dependencies

- express 
- dotenv
- mongoose
- jsonwebtoken
- bcryptjs
- cors

## start project

```bash
# clone repository
$ git clone https://github.com/maora96/provihack-back.git

# install Dependencies
$ npm install

# run app
$ npm start
```
